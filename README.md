# Conversor-mp3-wav

Conversor de áudio mp3 estéreo para wav mono em Python


 O código começa importando a classe AudioSegment.
 
 Em seguida, ele percorre todos os arquivos mp3 em um diretório e cria um objeto AudioSegment de cada um.
 
 O código então altera o número de canais para 1, os exporta como arquivos wav e os nomeia com sua extensão de arquivo mais ".wav".
